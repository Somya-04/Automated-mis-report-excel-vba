# 📊 Automated Sales MIS Report — Excel + VBA

![MIS Report Preview](mis-report-screenshot.png)

---

## 📌 Project Overview

A macro-enabled Excel MIS reporting tool that automatically generates
a fully formatted Monthly Sales MIS Report with a single button click.

Built to simulate real-world MIS Executive responsibilities —
data entry, report generation, KPI tracking, and PDF export —
all automated using VBA Macros.

---

## 🛠️ Tools & Technologies

| Tool | Purpose |
|------|---------|
| Microsoft Excel | Data storage and report output |
| VBA Macros | Report automation and formatting |
| Power Query | Data cleaning and transformation |
